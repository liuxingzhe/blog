yt-dlp-script介绍

yt-dlp-script是一个基于 yt-dlp 工具，实现了视频的单个或批量下载，兼容 Windows 与多种 Linux 发行版。用户只需按照提示操作，无需额外配置，就能轻松获取自己需要的视频文件。核心功能包括自定义 Cookie（支持登录受限视频）、指定输出目录、自动读取链接列表、以及在 Linux 下自动安装所需依赖。

yt-dlp-script截图界面

核心功能概览

脚本提供以下便利特性：

  • 一键启动：运行脚本后按照提示输入链接或文件路径，即可立即下载视频。
  • 自定义 Cookie：对需要登录验证的内容进行下载时，可将 Cookie 信息写入脚本。
  • 输出目录灵活指定：支持将下载内容保存到任意文件夹，便于管理和归档。
  • 批量下载模式：自动读取 urls.txt 文件，将其中所有链接依次抓取。
  • 自动依赖安装：Linux 环境下首次运行时，脚本会自动安装 yt-dlp 和 ffmpeg，无需手动干预。

Windows 平台使用指南

脚本在 Windows 下以批处理文件形式运行：

  1. 准备文件：将 yt-dlp.exe、ffmpeg.exe、ffprobe.exe 和 yt-dlp.bat 放在同一目录中。
  2. 执行脚本：双击 yt-dlp.bat，根据提示输入视频链接或批量文件路径。
  3. 下载结果：脚本完成后，可在指定目录下找到已下载的视频文件。

Linux 平台快速部署

支持 Debian、Ubuntu、Alpine、CentOS 等主流发行版。

bash -c "$(curl -fsSL https://raw.githubusercontent.com/nodeloc666/yt-dlp-script/main/install.sh)"

执行上述命令后，脚本会自动拉取最新版本并安装依赖。若要卸载：

bash -c "$(curl -fsSL https://raw.githubusercontent.com/nodeloc666/yt-dlp-script/main/uninstall.sh)"

应用场景与注意事项

  • 适合需要定期批量下载公开或内测视频的用户。
  • 在使用自定义 Cookie 功能时，请确保提供正确且合法的登录凭证。
  • 如果下载失败，可检查网络连接或更新 yt-dlp 版本。

这个一键脚本将视频下载流程极简化，结合灵活的自定义选项，为内容采集和离线存储提供了可靠方案。只需一次配置，后续使用便可无缝衔接，轻松应对各种下载需求。
